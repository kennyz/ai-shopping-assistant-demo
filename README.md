# CartPilot AI Shopping Assistant Demo

A static GitHub Pages demo for an AI shopping assistant. It simulates conversational product discovery, budget-aware ranking, comparison, shopping-cart state, and purchase insights entirely in the browser.

## Run Locally

Open `index.html` directly, or serve the folder:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Publish

This project is ready for GitHub Pages. Push it to a GitHub repository and enable Pages from the repository settings, or deploy from the `main` branch root.
